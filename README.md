# OCI_Language_AI

- Autonomous Database 23ai [Oracle Always Free Tier](https://www.oracle.com/au/cloud/free/) + APEX Workspace
- OCI Object Store Bucket
- API Key or OCI Resource Principal
- Permissions to use OCI AI Language Services
